1. 嚴選優質的納豆菌與紅麴菌種，利用現代生物科技發酵生產，提供菌體健康成長需要的養分基質及適宜的生長環境，使其同步產生多樣化有益於身體的營養成分。

2. 發酵產物未施以物理化學等萃取純化精製步驟，保留納豆與紅麴的營養價值與機能成分，成分組成自然且均衡。

3. 產品成分添加多種經尖端技術轉換之礦物質及微量元素，稱之為真原素 Primordia Elements®，可以補充身體正常生理功能運作所需的礦物質及微量元素，可促進新陳代謝及調節生理機能。

納 豆 紅 麴( Natto － Monascus )"順心"必備保健食品

素食起司 納豆

有"素食起司"之稱的納豆，是維持日本人健康長壽的秘訣之ㄧ。納豆是將煮熟黃豆直接接種納豆菌 ( Bacillus natto ) 所製成帶有黏性物質與獨特氣味的黃豆發酵食品。在中國明朝李時珍的「本草綱目」與日本江戶時代的「本朝食鑑」皆有記載納豆對於身體健康的好處。近代科學的研究成果，首推日本富崎醫科大學的須見洋行教授 ( Dr. Hiroyuki Sumi )，發現納豆中含有納豆激酶 ( nattokinase )。

納豆含有大豆胜肽、胺基酸、礦物質、維生素、皂素、卵磷脂、異黃酮等多種營養成分，為優質營養補給來源，有滋補強身的功效。納豆發酵過程中同時產生多種機能成分，包括：納豆激酶、維他命K2、ACEI等；其中的納豆激酶可以調節生理機能，使精神旺盛，有益健康。

中國御用養生珍品 紅麴

紅麴是紅麴菌 ( Monascus ) 生長於蒸煮米粒上發酵而成，是極珍貴的養生補品。在「日用本草」 ( 元。吳瑞 ) 、「飲膳正要」 ( 元。忽思慧 ) 、「本草衍義補遺」 ( 元。朱丹溪 ) 、「本草綱目」 ( 明。李時珍 ) 、「天工開物」 ( 明。宋應星 ) 、「本草備要」 ( 清。汪昂 ) 等歷代中國典籍都有記載紅麴對於身體健康的好處。近代科學研究發現，紅麴菌體生成的代謝產物含 monacolin K 等機能成分，因此紅麴曾被列為我國跨部會『保健食品研究開發計畫』的重點研究項目。

紅麴含有水解酵素、固醇類、異黃酮等多種營養成分，能調整新陳代謝，有助健康維持。紅麴發酵過程中同時產生多種珍貴的機能成分，包括： monacolin K 、γ - 胺基丁酸 ( GABA ) 、 dimerumic acid 、 flavonoids 等；其中的 monacolin K 可協助調整體質，維持健康。

支配人體生理機能正常運作 礦物質與微量元素

1992年在巴西舉行的「地球高峰會」中，一項由全球科學家共同發表的宣言中陳述"幾乎全球各地的土壤都普遍缺乏礦物質"。耕作農地土壤與水質的礦物質分佈將密切影響農作物中的礦物質含量；因此，這些栽種於貧脊土壤上的食物，已經無法提供身體所需足量且均衡的礦物質了。

長庚生技重建出原始蘊含於大自然中均衡完整的礦物質及微量元素，經與天然有機酸螯合處理，具有以下特點：

(1)具生物活性，有助於被生物組織吸收。

(2)補充身體所需的微量元素，可促進新陳代謝及調節生理機能。

(3)幫助調節身、心、靈轉化為和諧的健康狀態。

(4)真原素有助於生物機能成分之運作。

(5)提供多種經尖端技術轉換之礦物質及微量元素，稱之為真原素Primordia Element®是身體健康的關鍵因素。
