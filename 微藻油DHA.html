現代人飲食中ω-3脂肪酸攝取不足

脂肪是人體所需基本營養素，飲食內容需兼顧脂肪酸攝取的多樣化與均衡性，包含飽和脂肪酸、單元不飽和脂肪酸及多元不飽和脂肪酸，其中多元不飽和脂肪酸以ω-3及ω-6最為重要。

1994年聯合國糧農組織(FAO)及世界衛生組織(WHO)建議飲食中ω-3與ω-6之攝取比值為1：5；然而，大規模營養調查研究發現，現代人飲食模式偏向西化，導致ω-3與ω-6比值攀升至1：25，即ω-3過度缺乏而ω-6超量攝取，所造成的營養失衡會嚴重影響身體健康。

靈光的 ω -3脂肪酸-DHA

DHA是嬰幼兒及學童生長發育過程的關鍵營養素，也是孕婦及年長者必備的保健元素，營養科學研究證實DHA具有優越、多元的生理調節功能。DHA ( docosahexaenoic acid ) 為二十二碳六烯酸，是人體必需脂肪酸，其與EPA ( eicosapentaenoic acid ) 同屬ω-3長鏈多元不飽和脂肪酸，在體內無法自行合成，必須由飲食攝取。

深海魚油同時含有DHA及EPA，為攝取DHA的重要來源之一；針對素食者則可補充亞麻油，所含ω-3脂肪酸的α-次亞麻油酸 ( α-linolenic acid，ALA ) 在體內也能經由代謝轉換成部分DHA，補充生理機能運作之所需。歐美先進國家建議DHA的每日攝取量分別為：1~2歲幼兒70~140毫克，3歲以上兒童及成人90~200毫克，懷孕及哺乳期200毫克以上。

魚油DHA的問題點

市售DHA營養補充品最常見者為深海魚油，事實上，魚油中的DHA並非魚類生成，而是魚吃進含DHA之微藻後在魚體內蓄積；另外，目前魚油亦面臨著海洋生態、食品安全、使用禁忌等問題。在海洋生態方面，魚油為非永續資源，聯合國專家已提出警訊，除非能減少捕魚船隊使魚群得以復育，否則到2050年全球將出現海洋無魚之慘況。

在食品安全方面，大型魚種普遍有食物鏈累積重金屬(如甲基汞)及持久性環境汙染物(如多氯聯苯)殘留問題，其魚油製品即有重金屬及環境毒物殘留之疑慮，因此國內外食品法規有明訂魚油的多氯聯苯殘留限值。

在使用禁忌方面，魚油成分之高比例EPA可能使血流時間延長，舉凡懷孕後期婦女、凝血功能異常及服用抗凝血劑者，建議慎用魚油；同時，EPA以其結構相似之代謝競爭特性，還會抑制幼兒生長發育必需的關鍵脂肪酸 -DHA及花生四烯酸(ARA)之吸收。

優越的植物性微藻油DHA

在海洋魚類資源緊缺之際，選擇微藻油DHA是必然趨勢，其來源為取自單細胞微藻 -裂壺藻 ( Schizochytrium sp. ) 之微藻油。植物性微藻油是DHA的直接來源，沒有魚腥味，亦不含膽固醇。微藻在封閉式培養槽內以專利技術培育，無海洋食物鏈之重金屬或持久性環境汙染物殘留問題。

有別於高比例EPA之魚油，經由藻種篩選及培養條件控制，可取得EPA含量極低的微藻油DHA，無使血流時間延長之問題，亦不會阻礙幼兒對DHA及花生四烯酸(ARA)的吸收，因此更適合幼兒、懷孕及哺乳期婦女、凝血功能異常及服用抗凝血劑者食用。

值得一提的是，微藻油DHA為具有高人體吸收率之三酸甘油酯(TG)型式，優於經人工乙酯化使DHA轉為低吸收率之乙酯(EE)型式的濃縮魚油。
